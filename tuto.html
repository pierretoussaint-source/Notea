📖 Guide d'Utilisation : Newtea

Newtea est une application web progressive (PWA) conçue pour la création, la gestion et la notation d'essais agronomiques en plein champ. Elle a été pensée pour être rapide, intuitive, et surtout pour fonctionner 100% hors-ligne une fois installée sur votre appareil.

1. Installation sur Smartphone ou Tablette (Hors-Ligne)

Pour garantir que l'application fonctionne au milieu d'un champ sans aucun réseau internet, vous devez l'installer sur votre écran d'accueil.

  • Sur Android (Google Chrome) : Ouvrez le lien de l'application. Appuyez sur les trois petits points ⋮ en haut à droite du navigateur, puis sélectionnez "Ajouter à l'écran d'accueil".
  • Sur iOS (Apple Safari) : Ouvrez le lien de l'application. Appuyez sur l'icône "Partager" (le carré avec une flèche vers le haut) en bas de l'écran, faites défiler vers le bas, puis choisissez "Sur l'écran d'accueil".

Lancez ensuite l'application depuis la nouvelle icône apparue avec vos autres applications.

2. Démarrage et Configuration de la Session

À l'ouverture de l'application, l'écran d'accueil s'affiche. Suivez ces étapes :

  1. Entrez obligatoirement votre Nom / Code Notateur. Cela permettra d'identifier vos notes lors de l'export.
  2. Choisissez comment démarrer votre session :
    • Noter un essai existant : Sélectionnez un essai déjà en mémoire, puis cochez/décochez les Protocoles que vous souhaitez évaluer.
    • Importer un essai (CSV) : Chargez le plan d'un essai préparé sur Excel.
    • Créer un nouvel essai : Si vous n'avez pas de fichier, vous pouvez générer une grille vierge en indiquant le nom, l'espèce, les colonnes (X) et les lignes (Y).

3. L'Interface Principale & Navigation

L'application se divise en deux onglets principaux en haut de l'écran : Note (votre outil de travail principal) et Config (pour les paramètres et exports).

Comment sélectionner une parcelle ?

L'application vous propose deux méthodes extrêmement rapides :

Méthode 1 : Le Scanner QR Code

Idéal pour flasher directement l'étiquette de la parcelle et la valider instantanément.

Idéal pour voir où vous êtes (coordonnées X/Y) et cliquer sur la case suivante visuellement.

Le fond bleu désigne le type M (Mâle), le vert le type F (Femelle), et le jaune le type H (Hybride).
Lorsqu'une parcelle est hachurée, cela signifie qu'une note a déjà été enregistrée pour celle-ci ! La parcelle actuellement sélectionnée est en vert vif.

4. Saisir vos Observations

Une fois la parcelle sélectionnée (ou scannée via le QR Code), la parcelle est validée. L'interface de notation est déverrouillée.

  1. Ouvrez les menus via le bouton "► MENUS" et sélectionnez ce que vous souhaitez noter (Stade, Floraison, Paramètres, Photos...).
  2. Saisissez vos valeurs à l'aide des boutons + / -, ou en prenant une photo 📷.
  3. Ajoutez des Observations textuelles si nécessaire (vous pouvez utiliser le micro 🎤 pour dicter).
  4. Cliquez sur Enregistrer (ou Enr. Suivant) tout en bas pour sauvegarder. La note s'ajoutera dans le tableau des "Derniers enregistrements".

5. Fonctionnalités Expertes pour aller plus vite

🔲 La Saisie Groupée (Multi-Sélection)

Si plusieurs parcelles autour de vous ont exactement la même valeur (ex: un champ entier est au stade BBCH 65), utilisez la saisie groupée pour gagner un temps précieux :

Stade (BBCH) : 🔲
  1. Entrez la valeur "65" dans le champ Stade.
  2. Cliquez sur l'icône 🔲 située à côté du titre.
  3. Le grand plan de semis s'ouvre. Cliquez sur toutes les parcelles concernées (elles deviendront rouges). Astuce : cliquez sur un en-tête de colonne (X) ou de ligne (Y) pour sélectionner toute la ligne d'un coup !
  4. Cliquez sur le bouton final "Appliquer à la sélection". Les notes sont enregistrées.

⭐ Le Mode Zen (Plein Écran)

Vous ne voulez noter qu'un seul critère (ex: la floraison) et marcher rapidement dans le champ ? Activez le Mode Zen en cliquant sur l'étoile ⭐ à côté du titre de la notation.

Floraison (%) : ⭐

L'interface s'agrandira pour prendre tout l'écran, avec des boutons géants.

⚠️ TRÈS IMPORTANT EN MODE ZEN :
Dans ce mode, un bouton "⏭ Suivant" apparaît en haut. Il permet de passer directement à la parcelle suivante géographiquement... Mais il n'enregistre pas votre note !

Pour que la note soit sauvegardée, vous devez utiliser les boutons "Enregistrer" situés tout en bas de l'écran Zen.

6. Sauvegarde, Exportation et Sécurité

L'application ne nécessitant pas d'internet, absolument toutes vos données sont stockées à l'intérieur de votre navigateur internet. Elles ne sont envoyées sur aucun serveur.

  • Si vous videz l'historique ou le cache de votre navigateur, vos notes seront effacées.
  • Il est donc primordial d'exporter vos données régulièrement.

Allez dans l'onglet Configuration (Config) :

  1. Cliquez sur "📥/📤 Sauvegarder CSV, Partager & Purger".
  2. Le fichier contenant toutes vos notes va se télécharger automatiquement.
  3. Si votre appareil le permet, un menu de partage s'ouvrira (pour l'envoyer par email ou sur un Drive).
  4. Une boîte de dialogue vous demandera enfin si vous souhaitez purger (effacer) les notes de l'appareil maintenant que la sauvegarde est faite. Cliquez sur "OK" pour repartir sur une base propre le lendemain.

📖 User Guide: Newtea App

Newtea is a Progressive Web App (PWA) designed for creating, managing, and scoring agronomic field trials. It was built to be fast, intuitive, and most importantly, to work 100% offline once installed on your device.

1. Installation on Smartphone or Tablet (Offline Mode)

To ensure the application works in the middle of a field without any internet network, you must install it on your home screen.

  • On Android (Google Chrome): Open the app's link. Tap the three vertical dots ⋮ at the top right of the browser, then select "Add to Home screen".
  • On iOS (Apple Safari): Open the app's link. Tap the "Share" icon (the square with an upward arrow) at the bottom of the screen, scroll down, and choose "Add to Home Screen".

Then launch the application from the new icon that appeared with your other apps.

2. Startup and Session Configuration

When you open the app, the home screen appears. Follow these steps:

  1. You must enter your Name / Scorer Code. This identifies your scores when exporting.
  2. Choose how to start your session:
    • Score an existing trial: Select a trial already in memory, then check/uncheck the Protocols you want to evaluate.
    • Import a trial (CSV): Load a trial plan prepared on Excel.
    • Create a new trial: If you don't have a file, generate a blank grid by entering the name, species, columns (X), and rows (Y).

3. Main Interface & Navigation

The app is divided into two main tabs at the top of the screen: Note (your main working tool) and Config (for settings and exports).

How to select a plot?

Method 1: QR Code Scanner

4. Entering your Observations

Once the plot is selected (or scanned via QR Code), the plot is validated. This unlocks the scoring interface.

  1. Open the menus via the "► MENUS" button and select what you want to score (Stage, Flowering, Parameters, Photos...).

6. Backup, Export, and Security

Since the application does not require the internet, absolutely all your data is stored inside your internet browser. It is not sent to any server.

  • If you clear your browser's history or cache, your notes will be deleted.
  • It is therefore essential to export your data regularly.

Go to the Configuration (Config) tab:

  1. Click on "📥/📤 Save CSV, Share & Purge".
  2. The file containing all your notes will download automatically.
  3. If your device allows it, a share menu will open (to send it by email or to a Drive).
  4. Finally, a dialog box will ask if you want to purge (delete) the notes from the device now that the backup is done. Click "OK" to start with a clean slate the next day.

  1. Deschideți meniurile prin butonul "► MENIURI" și selectați ce doriți să evaluați (Stadiu, Înflorire, Parametri, Fotografii...).
  2. Introduceți valorile folosind butoanele + / - sau făcând o fotografie 📷.
  3. Adăugați Observații textuale dacă este necesar (puteți folosi microfonul 🎤 pentru dictare).
  4. Faceți clic pe Salvează (sau Salv. Următorul) în partea de jos pentru a salva. Nota se va adăuga în tabelul "Ultimele înregistrări".

5. Funcții Avansate pentru Rapiditate

🔲 Intrare în Grup (Selecție Multiplă)

Dacă mai multe parcele din jurul dvs. au exact aceeași valoare (ex: un câmp întreg este în stadiul BBCH 65), utilizați intrarea în grup pentru a economisi timp prețios:

Stadiu (BBCH) : 🔲
  1. Introduceți valoarea "65" în câmpul Stadiu.
  2. Faceți clic pe pictograma 🔲 de lângă titlu.
  3. Se deschide planul mare de semănat. Faceți clic pe toate parcelele relevante (ele vor deveni roșii). Sfat: faceți clic pe un antet de coloană (X) sau rând (Y) pentru a selecta toată linia deodată!
  4. Faceți clic pe butonul "Aplică la selecție". Notele sunt salvate imediat.

⭐ Mod Zen (Ecran Complet)

Doriți să evaluați doar un criteriu (ex: înflorirea) și să mergeți repede prin câmp? Activați Modul Zen făcând clic pe steaua ⭐ de lângă titlul notației.

Înflorire (%) : ⭐

Interfața se va extinde pentru a ocupa tot ecranul, cu butoane gigantice.

⚠️ FOARTE IMPORTANT ÎN MODUL ZEN:
În acest mod, un buton "⏭ Următorul" apare sus. Vă permite să săriți direct la parcela următoare din punct de vedere geografic... Dar nu salvează nota!

Pentru ca nota să fie salvată, trebuie să utilizați butoanele "Salvează" situate în partea de jos a ecranului Zen.

6. Copie de Rezervă, Export și Securitate

Deoarece aplicația nu necesită internet, absolut toate datele dvs. sunt stocate în interiorul browserului dvs. de internet. Ele nu sunt trimise către niciun server.

  • Dacă goliți istoricul sau memoria cache a browserului, notele dvs. vor fi șterse.
  • De aceea este esențial să vă exportați datele în mod regulat.

Mergeți la fila Configurare (Config):

  1. Faceți clic pe "📥/📤 Salvare CSV, Partajare și Golire".
  2. Fișierul conținând toate notele dvs. se va descărca automat.
  3. Dacă dispozitivul permite, se va deschide un meniu de partajare (pentru a-l trimite prin email sau pe Drive).
  4. În final, o casetă de dialog vă va întreba dacă doriți să goliți (ștergeți) notele de pe dispozitiv acum că a fost făcută copia de rezervă. Faceți clic pe "OK" pentru a începe pe curat a doua zi.

📖 Посібник користувача: Newtea

Newtea — це прогресивний веб-додаток (PWA), призначений для створення, управління та оцінювання агрономічних польових випробувань. Він створений швидким, інтуїтивно зрозумілим і, головне, для роботи на 100% офлайн після встановлення на ваш пристрій.

1. Встановлення на смартфон або планшет (Офлайн-режим)

Щоб програма гарантовано працювала посеред поля без доступу до Інтернету, ви повинні встановити її на головний екран.

  • На Android (Google Chrome): Відкрийте посилання на програму. Торкніться трьох вертикальних крапок ⋮ у верхньому правому куті браузера, потім виберіть "Додати на головний екран".
  • На iOS (Apple Safari): Відкрийте посилання на програму. Торкніться значка "Поділитися" (квадрат зі стрілкою вгору) внизу екрана, прокрутіть вниз і виберіть "На початковий екран".

Потім запустіть програму з нової іконки, що з'явилася разом з іншими вашими програмами.

2. Запуск та Налаштування сесії

Після відкриття програми з'явиться головний екран. Виконайте ці кроки:

  1. Вам необхідно обов'язково ввести Ім'я / Код Оцінювача. Це дозволить ідентифікувати ваші оцінки під час експорту.
  2. Виберіть, як почати сесію:
    • Оцінити існуюче випробування: Виберіть випробування, що вже збережене в пам'яті, потім виберіть/зніміть вибір з Протоколів, які ви хочете оцінити.
    • Імпорт випробування (CSV): Завантажте план випробування, підготовлений в Excel.
    • Створити нове випробування: Якщо у вас немає файлу, створіть порожню сітку, вказавши назву, вид, колонки (X) та рядки (Y).

3. Головний інтерфейс та Навігація

Додаток розділений на дві головні вкладки у верхній частині екрана: Нотатка (ваш основний робочий інструмент) та Конфігурація (налаштування та експорт).

Як вибрати ділянку?

Програма пропонує два надзвичайно швидкі методи:

Метод 1: Сканер QR-кодів

Ідеально підходить для прямого сканування етикетки ділянки та миттєвого підтвердження.

📷
Метод 2: Міні-план

Ідеально, щоб бачити, де ви знаходитесь (координати X/Y), і візуально вибрати наступний квадрат.

Y\X123
1101102103
💡 Кольорові коди плану:
Синій фон позначає тип M (Чоловічий), зелений — тип F (Жіночий), а жовтий — тип H (Гібрид).
Якщо ділянка заштрихована, це означає, що оцінка для неї вже записана! Поточна вибрана ділянка підсвічується яскраво-зеленим.

4. Введення спостережень

Після вибору ділянки (або сканування за допомогою QR-коду) ділянка підтверджується. Це розблокує інтерфейс оцінювання.

  1. Відкрийте меню за допомогою кнопки "► МЕНЮ" і виберіть, що ви хочете оцінити (Стадія, Цвітіння, Параметри, Фотографії...).
  2. Введіть значення за допомогою кнопок + / - або зробивши фотографію 📷.
  3. Додайте текстові Примітки за потреби (можна використовувати мікрофон 🎤 для диктування).
  4. Натисніть Зберегти (або Збер. Далі) у самому низу, щоб зберегти. Нотатка додасться до таблиці "Останні записи".

5. Розширені функції для прискорення роботи

🔲 Групове введення (Множинний вибір)

Якщо кілька ділянок навколо вас мають абсолютно однакове значення (напр., ціле поле на стадії BBCH 65), використовуйте групове введення, щоб заощадити дорогоцінний час:

Стадія (BBCH) : 🔲
  1. Введіть значення "65" у поле Стадія.
  2. Натисніть іконку 🔲 поруч із назвою.
  3. Відкриється великий план посіву. Клацніть на всі відповідні ділянки (вони стануть червоними). Порада: натисніть на заголовок колонки (X) або рядка (Y), щоб вибрати всю лінію одразу!
  4. Натисніть кнопку "Застосувати до виділеного". Оцінки будуть миттєво збережені.

⭐ Режим Zen (Повноекранний)

Хочете оцінювати лише один критерій (напр., цвітіння) і швидко йти по полю? Увімкніть режим Zen, натиснувши зірочку ⭐ поруч із назвою оцінки.

Цвітіння (%) : ⭐

Інтерфейс розшириться на весь екран із гігантськими кнопками.

⚠️ ДУЖЕ ВАЖЛИВО В РЕЖИМІ ZEN:
У цьому режимі зверху з'являється кнопка "⏭ Наступний". Вона дозволяє перейти безпосередньо до наступної географічно ділянки... Але вона не зберігає вашу оцінку!

Щоб оцінка збереглася, ви повинні використовувати кнопки "Зберегти", розташовані в самому низу екрана Zen.

6. Резервне копіювання, Експорт та Безпека

Оскільки програма не потребує Інтернету, абсолютно всі ваші дані зберігаються всередині вашого браузера. Вони не надсилаються на жоден сервер.

  • Якщо ви очистите історію або кеш вашого браузера, ваші нотатки будуть видалені.
  • Тому дуже важливо регулярно експортувати свої дані.

Перейдіть на вкладку Конфігурація (Config):

  1. Натисніть на "📥/📤 Зберегти CSV, Поділитися та Очистити".
  2. Файл з усіма вашими нотатками завантажиться автоматично.
  3. Якщо ваш пристрій дозволяє, відкриється меню обміну (щоб надіслати по email або на Drive).
  4. Нарешті, діалогове вікно запитає, чи хочете ви очистити (видалити) нотатки з пристрою, оскільки резервна копія зроблена. Натисніть "ОК", щоб почати з чистого аркуша наступного дня.
